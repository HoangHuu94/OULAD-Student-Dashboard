# Educational Management Dashboard (OULAD Dataset)

## Project Overview
This project focuses on analyzing student engagement and academic performance patterns using the Open University Learning Analytics Dataset (OULAD). The goal is to provide actionable insights to improve student retention and academic success.

## Tech Stack
* **Visualization Tool:** Power BI
* **Database/Source:** OULAD Dataset (CSV files)
* **Techniques:** Data Modeling, ETL, DAX, Data Visualization

## Key Contributions & Data Flow
* **End-to-End Data Modeling:** Engineered a robust 7-table star schema to connect student demographics, registrations, assessments, and VLE (Virtual Learning Environment) interaction logs.
* **ETL Pipeline:** Performed data cleaning and transformation to handle missing values and ensure data consistency across multiple sources.
* **Insightful Visualization:** Developed interactive dashboards with filters for courses and semesters, allowing for deep dives into student behavior.

## Key Insights & Business Impact
* Identified "at-risk" student groups by analyzing early engagement signals with learning materials.
* Visualized the correlation between assessment scores and final academic outcomes.
* Provided data-driven recommendations for academic advisors to intervene early and support struggling students.
